What is MUM?

Work in progress

Not all information is included in the docs yet, but we are working on it!

MUM is a user management web interface for email servers. It is designed to be easy to use but still have powerful features.

The software works with the popular open source packages Postfix and Dovecot that can use MUM's MySQL database for their lookups. We provide all of the configuration that is necessary. Additionally, you may integrate any other software with MUM's workflow using our integrations. This makes MUM a super flexible tool that is compatible to most environments.
